
# テニスコート予約管理アプリ 開発ガイド（README）

## 🎾 概要

このリポジトリは、テニスコートの抽選・予約情報を共有・管理するための
Streamlitアプリケーションです。
見た目がポップで、スマートフォンからも簡単に操作できます。

## 🖥 開発環境セットアップ

### 1️⃣ リポジトリをローカルにクローン

PowerShell またはターミナルで以下を実行：

```bash
# 任意の作業フォルダへ移動
cd C:\Users\あなたの名前\Documents

# GitHubからクローン（URLはあなたのリポジトリURLに置き換え）
git clone https://github.com/あなたのGitHubユーザー名/テニスコート予約管理.git

cd テニスコート予約管理
```
